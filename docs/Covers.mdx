## About

Covers are machine attachments that serve as utilities for acomplishing different tasks.
They're designed to be faster & more effective than traditional methods, great for automation and saving space.

Nearly all GregTech machines are coverable.
Covers usually interact with the machine's inventory and the inventory they're facing.
They can be configured using a Screwdriver, and removed using Crowbars.

### Not everything comes at a price
You can get a **free** cover that blocks all interactions by right-clicking a machine with a screwdriver.
Right-click again to get one that only blocks redstone signal.

### Redstone Covers

| Type | Effect | Assembler Recipe
|:----:|:----:|:----:|
| Item Detector | Outputs redstone signal depending on stored items | 1x Weighted Pressure Plate (Heavy), 1x Iron/Aluminium Plate |
| Liquid Detector | Outputs redstone signal depending on stored liquids | 1x Weighted Pressure Plate (Light), 1x Iron/Aluminium Plate |
| EU Detector | Outputs redstone signal depending on stored energy | 1x EU-Reader, 1x Iron/Aluminium Plate |
| Signalizer | Applies a constant redstone signal to the machine | 1x Redstone Torch, 1x Iron/Aluminium Plate |
| Machine Controller | Allows enabling/disabling machines through redstone signal | 1x Lever, 1x Iron/Aluminium Plate |
| Active Machine Detector | Outputs redstone signal if the machine has work | 1x Redstone Comparator, 1x Iron Plate <br/> OR 1x Aluminium Plate, 1x Redstone Torch |
| Conductor | Throughputs the strongest redstone signal to the cover facing | 1x Redstone, 1x Iron Plate |
